本项目后续生成的文档全部使用中文，包括 Trellis 的 prd.md、design.md、implement.md 以及其他交付文档。

**Why:** 用户明确要求。项目默认语言为中文，文档保持一致。

**How to apply:** 创建或更新 `.trellis/tasks/` 下的任何文档时，使用中文撰写。代码注释和技术术语可保留英文原文（如 fixture、dry-run、Keychain 等）。
